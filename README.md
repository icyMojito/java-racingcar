### 기능 요구 사항

- 입력값이 빈 문자열 또는 null 값이면 0을 반환한다.
- 입력값이 숫자 이외의 값 또는 음수이면 RuntimeException 예외가 발생한다.
- 쉼표(,) 또는 콜론(:)으로 문자열을 나눈다.
- '//'와 '\n' 사이의 문자를 커스텀 구분자로 추출한다.
- '\n' 이후의 문자열을 커스텀 구분자로 분리한다.
- 나눈 문자열의 요소들을 숫자로 변환한다.
- 숫자로 변환한 요소들을 모두 합한다.
- 모두 합한 결과를 반환한다.
